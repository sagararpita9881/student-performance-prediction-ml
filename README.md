# Student Performance Prediction Using Machine Learning

## Project Overview
This project predicts the final CGPA of students using academic and behavioral data.

## Dataset
The dataset includes:
- Gender
- Age
- Major
- Attendance percentage
- Study hours per day
- Previous GPA
- Sleep hours
- Social hours per week

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn

## Machine Learning Model
- Linear Regression

## Evaluation Metrics
- Mean Absolute Error (MAE)
- R² Score

## Results
- MAE ≈ 0.13
- R² Score ≈ 0.88

## Conclusion
Previous GPA and study hours were the most important factors affecting student performance.
